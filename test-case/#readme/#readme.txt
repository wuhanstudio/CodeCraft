本用例使用说明：
	1.本数据非官方，仅供算是调试使用，检查程序是否有bug等;
	2.本数据提供为方便交流，例如算法调试某用例是否有解，耗时，当前最优解;
	3.数据在下已经跑过，70%左右有解，case/文件夹中，有形如;"权值_331_result.csv"的文件，表示权值为331。当权值为100000时，表示该样例无解（此无解是在下测试的，并不一定有解）。
	4.交流时可用 N-nci 表示用例，即600-50c2 表示600个顶点50个条件点中第二个样例！
